# hello-phaser-webpack 

The minimum project for writing Phaser 3 with modern ES.

- Phaser 3
- webpack 4
- ESLint
- Prettier

## Scripts

- `yarn start`: Start dev server.
- `yarn build`: Build for production.
- `yarn lint`: Run lint.
- `yarn fix`: Auto fix with Lint.
